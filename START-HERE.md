# START-HERE — мастер-класс: твой первый бот за час

> **Зачем это нужно.** Это короткий вход для участника. Ты откроешь пакет в Claude Code, вставишь один промпт (готовый текст-команду) — и Claude проведёт тебя **за руку** к работающему боту. Здесь только старт; каждый следующий шаг Claude объяснит сам.

---

## ✅ Что должно быть готово (проверь за минуту)

Всё это ты подготовил по письму-приглашению. Быстрая самопроверка:

- [ ] **Claude Code** — активная **платная подписка** (Pro/Max или Console) и ты умеешь им пользоваться (CLI или приложение). Бесплатный чат claude.ai не подходит.
- [ ] **Токен Telegram-бота** — получен у **@BotFather** (это встроенный сервис Telegram для создания ботов).
- [ ] **Username в Telegram** задан (`@имя`) — по нему бот проверяет доступ.
- [ ] **Ключ AssemblyAI** — из твоего личного кабинета AssemblyAI (сервис распознавания речи).
- [ ] **Доступ к n8n** — ты зашёл по приглашению на почту, и у тебя есть свой **API-ключ n8n** (секретная строка доступа; как создать — в письме).
- [ ] **Адрес n8n** — ссылку на инстанс даст ведущий.

Чего-то нет — скажи ведущему или помощнику **сейчас**, не жди начала практики.

> Пакет рассчитан и протестирован на **Claude Code**. Другой агентный инструмент с поддержкой MCP (напр. Codex CLI) в принципе может подойти, но стартовый промпт и `.mcp.json` заточены под Claude Code — **совместимость с аналогами не гарантируется**.

## Шаг 1. Создай папку и открой в ней Claude Code

- Создай новую **пустую папку** для проекта — там же, где ты обычно держишь проекты Claude Code (чтобы у него был к ней доступ). Назови как удобно, напр. `bot-workshop`.
- Открой **Claude Code в этой папке**.

Больше ничего вручную качать не нужно: **пакет заберёт сам Claude** по стартовому промпту (Шаг 2).

> Git и SSH-ключи тебе **не нужны** — Claude скачает пакет сам (архивом, если git не установлен).

## Шаг 2. Вставь стартовый промпт

Скопируй этот текст **целиком** и отправь Claude **первым сообщением** (в пустой папке):

```
Ты — Claude Code в РЕЖИМЕ МАСТЕР-КЛАССА (учебный воркшоп, ~1 час).

Папка сейчас пустая. Сначала забери в неё учебный пакет из репозитория:
https://github.com/sergeykruglovit-a11y/bot-starter-workshop
Скачай его содержимое в ТЕКУЩУЮ папку любым надёжным способом (git не
обязателен — если его нет, скачай и распакуй архив репозитория). Убедись,
что рядом появились файлы CLAUDE-workshop.md и START-HERE.md.

Затем прочитай CLAUDE-workshop.md и веди меня СТРОГО по нему:
- объяви режим мастер-класса и назови задачу (Telegram-бот: голосовое →
  распознавание AssemblyAI → текст ответом, только разрешённым по списку);
- предупреди, что путь УПРОЩЁННЫЙ (брифинг уже зашит, сбор требований не нужен);
- дальше — один шаг за раз, жди моего «готово» после каждого;
- workflow создавай active:false — активирую и тестирую я сам;
- секреты в чат НЕ проси: ключ n8n я впишу в .mcp.json сам, токен Telegram и
  ключ AssemblyAI заведу в credentials n8n сам — ты только называешь, что создать.

Начни с приветствия и подключения пакета.
```

## Шаг 3. Дальше — следуй за Claude

Claude объявит режим мастер-класса и поведёт по шагам. **Твоя роль:** делать по одному действию и писать «готово» — тогда идёт следующий шаг. Не торопись вперёд.

## 🔑 Куда пойдут твои ключи (и почему не в чат)

| Ключ | Куда вводишь | Кто вводит |
|------|--------------|:----------:|
| API-ключ **n8n** | в файл `.mcp.json` (на твоём компьютере) | ты сам |
| Токен **Telegram** | в **credentials n8n** (в интерфейсе n8n) | ты сам |
| Ключ **AssemblyAI** | в **credentials n8n** | ты сам |

**Никогда не отправляй ключи в переписку с Claude.** Всё, что попадает в чат, уходит в модель. Claude о ключах в чат не просит — он только подсказывает, куда их вписать.

## Что мы построим

Голосовое в Telegram → проверка доступа по списку → распознавание в AssemblyAI → текст ответным сообщением.

Маршрут часа: **связь с n8n → ключи в n8n → таблица доступа (твой `@username`) → бот собирается → активация → тест голосовым**.

## Не успел за час?

Ничего страшного — **пакет остаётся у тебя**. Файл `CLAUDE-workshop.md` рядом — это полная инструкция; продолжишь с любого места как домашнее задание, и Claude так же поведёт тебя за руку.
