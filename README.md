# Мастер-класс: Telegram-бот транскрипции голоса на n8n + Claude Code

Самодостаточный учебный набор для **часового практического мастер-класса**. За час участник-новичок **своими руками** (силами ИИ-агента **Claude Code**) собирает Telegram-бота, который принимает голосовое сообщение, распознаёт речь через **AssemblyAI** и присылает текст ответом — только разрешённым пользователям (доступ по списку в **n8n Data Table**).

> Это учебная версия с **одной зашитой задачей** и путём «за руку». Для реальной автоматизации своих задач существует полноценная сборка n8n-бота (проектирование под конкретную задачу через сбор требований) — этот пакет с ней не связан и живёт самостоятельно.

## Кому какой файл

| Файл | Для кого | Что это |
|------|----------|---------|
| **`START-HERE.md`** | участник | С чего начать: чек-лист готовности, стартовый промпт для Claude Code, карта часа |
| **`CLAUDE-workshop.md`** | Claude Code | Зашитый брифинг: агент читает его и ведёт участника по шагам |
| **`.mcp.json.example`** | участник | Образец конфига MCP (копируется в `.mcp.json`, туда впишешь свой ключ n8n) |

Участник: открой пакет в Claude Code и следуй **`START-HERE.md`** — дальше Claude ведёт за руку.

> Материалы для ведущего и организатора (план часа, шаблон приглашения, чек-листы готовности) поставляются организатором отдельно и в этот пакет для участника не входят.

## Что нужно участнику (готовится заранее, ~15 минут)

- **Claude Code** — установлен и открывается.
- Доступ к **n8n** (даёт организатор) и **свой API-ключ n8n**.
- **Токен Telegram-бота** (через @BotFather) и **ключ AssemblyAI** (бесплатный тариф).

Точные шаги пред-работы придут в письме-приглашении от организатора. Git и SSH-ключи **не нужны** — пакет забирается по открытой ссылке.

## Безопасность

- Ключи **никогда не отправляются в чат** с Claude. Ключ n8n — в файл `.mcp.json`; токен Telegram и ключ AssemblyAI — в credentials n8n.
- `.mcp.json` — в `.gitignore`, в репозиторий не попадает. Настрой его из образца:

```bash
cp .mcp.json.example .mcp.json   # затем впиши свой хост и ключ n8n
```

## Требования к площадке (для организатора)

- **n8n доступен из интернета** (публичный HTTPS) — Telegram-бот работает через webhook, локальный n8n его не примет.
- У приглашённых пользователей есть права создавать workflow, credentials, Data Table и свой API-ключ.
