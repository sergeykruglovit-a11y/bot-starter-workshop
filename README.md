# Мастер-класс: Telegram-бот транскрипции голоса на n8n + Claude Code

Самодостаточный учебный набор: за ~час ты **своими руками** (силами ИИ-агента **Claude Code**) собираешь Telegram-бота, который принимает голосовое сообщение, распознаёт речь через **AssemblyAI** и присылает текст ответом — только разрешённым пользователям (доступ по списку в **n8n Data Table**). Опыт в программировании не нужен — бота собирает агент, ты ведёшь его по шагам.

> Это учебная версия с **одной зашитой задачей** и путём «за руку». Для реальной автоматизации своих задач существует полноценная сборка n8n-бота (проектирование под конкретную задачу через сбор требований) — этот пакет с ней не связан и живёт самостоятельно.

## Кому какой файл

| Файл | Для кого | Что это |
|------|----------|---------|
| **`START-HERE.md`** | участник | С чего начать: чек-лист готовности, стартовый промпт для Claude Code, карта часа |
| **`CLAUDE-workshop.md`** | Claude Code | Зашитый брифинг: агент читает его и ведёт участника по шагам |
| **`.mcp.json.example`** | участник | Образец конфига MCP (копируется в `.mcp.json`, туда впишешь свой ключ n8n) |

С чего начать: следуй **`START-HERE.md`** — создаёшь пустую папку и вставляешь стартовый промпт, дальше Claude сам скачивает пакет и ведёт за руку.

## Что нужно (подготовить заранее)

**Предусловие:** активная **платная подписка Claude Code** (Pro/Max или Console) и навык работы с ним (CLI или приложение) — не бесплатный чат claude.ai. Плюс установленный **Node.js (LTS)** — на нём работает связка Claude ↔ n8n.

> Вместо Claude Code теоретически можно взять другой агентный инструмент с поддержкой MCP (например, Codex CLI), но пакет протестирован только на Claude Code — совместимость с аналогами не гарантируется.

Заранее (~20–30 минут):
- **Node.js (LTS)** — [nodejs.org](https://nodejs.org). Без него бот не соберётся (самый частый затык).
- **Свой n8n**, доступный из интернета: проще всего облачный **n8n Cloud** (бесплатный пробный период) — или self-hosted **Community Edition** с публичным HTTPS-адресом. Плюс **свой API-ключ n8n** (Settings → n8n API).
- **Токен Telegram-бота** (через @BotFather) и заданный **username** в Telegram.
- **Ключ AssemblyAI** ($50 бесплатного кредита, без карты); запомни свой **регион** (EU/US).

Пошаговый чек-лист и стартовый промпт — в **`START-HERE.md`**. Git и SSH-ключи **не нужны** — пакет забирается по открытой ссылке.

## Безопасность

- Ключи **никогда не отправляются в чат** с Claude. Ключ n8n — в файл `.mcp.json`; токен Telegram и ключ AssemblyAI — в credentials n8n.
- `.mcp.json` — в `.gitignore`, в репозиторий не попадает. Настрой его из образца:

```bash
cp .mcp.json.example .mcp.json   # затем впиши свой хост и ключ n8n
```

## Требования к n8n

- **n8n доступен из интернета** (публичный HTTPS) — Telegram-бот работает через webhook, локальный n8n его не примет. Облачный n8n Cloud подходит из коробки.
- У твоего пользователя n8n есть права создавать workflow, credentials, Data Table и свой API-ключ (в своём аккаунте — по умолчанию).
